#🔐 Login Page UI

📌 Project Overview
This project is a simple and clean login page interface designed using HTML and CSS. The focus is on creating a modern and user-friendly UI.

🚀 Features
- Email input field
- Password input field
- Login button
- Clean and minimal design

 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

📂 Project Structure
- login.html → Structure of login page
- login.css → Styling of the UI
- login.js


📖 Learning Outcomes
- UI design fundamentals
- Form creation using HTML
- CSS alignment and spacing

📌 Conclusion
This project improved my UI design skills and helped me understand how to create simple yet effective user interfaces.
